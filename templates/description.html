Matériel

Un microcontrôleur ATMEL communique avec le Raspberry Pi en série via un protocole I2C. C'est le microcontrôleur des Arduino, il permet une interface avec des capteurs numériques ou analogiques. Il est monté sur une carte munie de connecteurs Grove de Seeeduino qui propose une large gamme de capteurs compatibles. Le microcontrôleur compatible Arduino bénéficie de son environnement de développement et des nombreuses bibliothèques de fonctions disponibles.

Logiciel

Le contrôle et la récupération des données des capteurs se fait via une interface web. Le serveur est programmé en python avec le microframework Flask. Le rafraichissement des données des capteurs se fait grâce à un script jQuery qui effectue des requêtes à intervalles de temps réguliers. Le streaming vidéo se fait par Flask qui implémente cette fonctionnalité.
